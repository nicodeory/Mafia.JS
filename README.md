# Mafia.JS
A work-in-progress NodeJS-based sc2 mafia clone. It will try to mimic every mechanic from the original Arcade Gamemode.

Some of the roles already implemented:
- Godfather (and the Triad equivalent)
- Mafioso (and the Triad equivalent)
- Blackmailer
- Citizen
- Crier
- Doctor
- Escort
- Sheriff
- Lookout
- Detective
- Vigilante
